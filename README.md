# PayPilot App

Next.js app for PayPilot.

## Run On Desktop (default)

```bash
npm run dev
```

## Run Safe Copy On Modal CPU

```bash
npm run dev:safe
```

This runs the same `npm run dev` flow in Modal and prints a tunnel URL for port `3000`.

## Build / Start / Lint

Desktop:

```bash
npm run build
npm run start
npm run lint
```

Safe (Modal):

```bash
npm run build:safe
npm run start:safe
npm run lint:safe
```
